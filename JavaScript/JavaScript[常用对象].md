#### 历史记录

window.history.$item

$item取值：

- length:历史记录URL个数
- forward():回退页面后，使用此方法可返回之前页面等价于下面一项
- go(parm):回退页面，使用此方法可返回指定，parm指定的位置是相对于当前页面的，当前页面为0，返回为负数，前进为正数
